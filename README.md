# Movie Recommender System

## Overview

This project is a content-based movie recommendation system built using Machine Learning techniques.

Given a movie, the system recommends similar movies based on genres, keywords, cast, crew, and overview information.

## Features

* Content-based filtering
* Movie recommendations using Cosine Similarity
* Text vectorization using CountVectorizer
* Streamlit-based user interface

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* NLTK
* Streamlit

## Dataset

TMDB 5000 Movies Dataset

## How It Works

1. Data preprocessing and cleaning
2. Feature engineering using genres, keywords, cast, crew, and overview
3. Creation of a combined tags column
4. Vectorization using CountVectorizer
5. Similarity computation using Cosine Similarity
6. Recommendation of top similar movies

## Future Improvements

* Movie poster integration using TMDB API
* Deployment on Streamlit Cloud
* Hybrid recommendation system
